# SEO King - 关键词分析系统

一个用于SEO关键词分析的Web应用，帮助分析竞品关键词、识别市场机会。

## 功能特点

- 多Excel文件上传与解析
- 自动检测重复词和品牌流量词
- 智能关键词分类（市场关键词、通用关键词、领域词）
- 气泡图可视化市场机会
- CSV导出功能

## 在线使用

访问: https://saul-ye3011.github.io/seo-king/

## 本地运行

```bash
npm install
npm run dev
```

## 技术栈

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- xlsx.js
